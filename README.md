# PSPF Explorer

An interactive compliance dashboard for the Australian Government Protective Security Policy Framework (PSPF) 2025 requirements. It surfaces domain summaries, heatmaps, and the ability to track progress by tagging requirements.

## Getting started
1. Click the link

All data is stored locally in the browser, so you can experiment without anything going anywhere or being stored on a server.

2. Import date if you have some, if not, start updating.
3. Export to backup, or to share your data with a friend.
